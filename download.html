Download

The installation file includes the Full version of Alpaca4d. Make sure you have Rhino3D installed.

downloadAlpaca4d

Setting Up

  1. Unblock the .zip file
  2. Open Rhino and Grasshopper
  3. Open the Library Folder
    1. (Grasshopper -> File -> SpecialFolder -> Component Folder)
  4. Copy the folder "Alpaca4d" in the Grasshopper Libraries folder
  5. Open the User Object Folder
    1. (Grasshopper -> File -> Special Folder -> User Object Folder)
  6. Copy the CONTENT of the UserObjects folder into Grasshopper User Object Folder
  7. Restart Rhino and Grasshopper.
