Tutor: Marco Pellegrino
Location: Florence (Italy)
Time: January 20-21, 9.30 am - 5.30 pm

This three day workshop is intended for beginner to intermediate users of Alpaca4d. Marco will explore with you more some techniques of working with complex parametric structural models through scripting using Grasshopper.

Furthermore, the user will learn how to navigate the source code and implement new elements and grasshopper components.
